# 🏥 Sistema de Gestão de Consultas

## 📌 Descrição

Sistema desenvolvido para gerenciamento de consultas médicas, permitindo o controle de médicos, pacientes e agendamentos.

---

## ⚙️ Tecnologias

* TypeScript
* JavaScript

---

## 🚀 Funcionalidades

* Cadastro de médicos
* Cadastro de pacientes
* Agendamento de consultas
* Controle de status das consultas

---

## 📂 Estrutura

O projeto utiliza interfaces para organizar os dados principais:

* Médico
* Paciente
* Consulta
* Especialidade
* Usuário Administrador
